マップアート作成ツールの使い方

1. 画像をアップロード

  • 「画像を選択」ボタンをクリックして、マップアートにしたい画像をアップロードします。
  • 対応形式は PNG / JPEG などです。 変換できなかったら画像を対応した形式にしてみてください。

2. 使用するブロックを設定

  • 使用するブロックの種類を選択します。
  • 使用するブロックを選択出来たら「ドットに変換」ボタンを押し、編集へ進んでください。

3. 編集する

  • 生成されたマップアートを編集することができます（ブロックの置き換え、塗りつぶし、左右反転など）。
  • 拡大・縮小やスクロール機能で細部まで編集が可能です。
  • 編集が終わったらコマンド生成ボタンを押してください

4. マップアートを出力する生成する平面と座標を決める

  • 「コマンド生成」ボタンをクリックして、マップアートを生成する平面の設定と座標の設定をします。
  • マップアートを生成する平面を設定します。(z-x面は地面と平行にマップアートを生成、x-y面,y-z面は地面と垂直にマップアートを生成します。)
  • マップアートを設置する座標（x, y, z）を指定します。(マップアートが生成される範囲のうち、x,y,zが一番小さい座標を指定してください。)

5. ダウンロードされたフォルダの中身をMinecraftのフォルダに移動させる

  • 生成されたデータはZIPフォルダとしてダウンロードされます。
  • ZIPフォルダの中の「dot」という名前のフォルダをMinecraftの「datapacks」フォルダに移動させます。
  • 「datapacks」フォルダはデフォルトでは「C:\Users\(ユーザー名)\AppData\Roaming\.minecraft\saves\(ワールド名)\datapacks」という位置にあります。

6. コマンドを実行する

  • Minecraft内で 「/reload」 を実行してマップアート生成コマンドを読み込みます。
  • 「/reload」を実行した後、「/function dot_command:command」を実行するとマップアートが生成されます。
  • (ファイルの場所が適正な位置に配置されていれば「/function」を打った時点でコマンドの予測変換に表示されるはずです)
